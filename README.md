## 📦 Getting Started

```bash
# Clone the repository
git clone https://github.com/mehergautam/Client-Project--3.git

# Navigate to project
cd Client-Project--3

# Install dependencies
npm install

# Run development server
npm run dev

# Open in browser
http://localhost:3000
```
